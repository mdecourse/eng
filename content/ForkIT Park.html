ForkIT Park

ForkIT Informechatronic Virtual Park - ForkIT 機電資整合模擬園區 

2021 Spring:

利用參數 CAD 建立機電整合零組件, 組立後轉入 CoppeliaSim 進行基本 Kinematic 模擬後, 利用 Lua Script 執行初步控制, 接著嘗試透過 Python Remote API 進行元端控制, 並設法利用 Image Sensor 將模擬場景 Streamming 至 Flask 伺服程式後, 嘗試在遠端電腦, 平板或手機瀏覽器, 甚或在手機 APP 上進行 Monitor 與操控.

相關技術:

在 Onshape, Solvespace, SolidWorks, Inventor, NX12 上建立機電整合系統零組件 (電腦輔助設計實習課程中, 基本 3D 模型設計繪圖能力).

從 CAD 中以 STL 轉出後, 進入 CoppeliSim 進行場景配置, 以 Lua Script 進行初步控制模擬 (計算機程式課程中, 了解 Python 與 C 基本語法後, 透過網際 Lua 環境 以及 http://mde.tw/cad2020/content/CoppeliaSim.html 中的範例程式, 了解如何編寫 CoppeliaSim Script 進行模擬監控).

以 Python Remote API 進行遠端控制模擬 (計算機程式課程內容)

以 Flask 將 Image Sensor 影像串流至 client 端瀏覽器 (網際內容管理與協同產品設計實習課程內容, 參考: self-driving car, Airhockey)

以 Flutter 建立手機 APP 與 Flask 伺服器互動, 取得串流場景並進行遠端 Monitor 與操控 (計算機程式課程延伸: http://mde.tw/cp2019/content/index.html)

2021 Fall 起除了 CoppeliaSim 外將要導入 RoKiSim, RoboDK 與 Webots 進行相關 Kinematic 模擬與 Dynamic 模擬.
